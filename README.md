# Humanizer UK

Skill для Claude Code, який видаляє ознаки AI-генерації з українського тексту, робить його живим і людяним.

## Встановлення

### Рекомендований спосіб
```bash
git clone https://github.com/StNatalia/humanizer-uk.git ~/.claude/skills/humanizer-uk
```

### Ручне встановлення
```bash
mkdir -p ~/.claude/skills/humanizer-uk
curl -o ~/.claude/skills/humanizer-uk/SKILL.md https://raw.githubusercontent.com/StNatalia/humanizer-uk/main/SKILL.md
```

## Використання

В Claude Code:
```
/humanizer-uk
[вставте ваш текст]
```

Або просто попросіть:
```
Олюдни цей текст: [ваш текст]
Прибери воду: [ваш текст]
Зроби живим українською: [ваш текст]
```

## 23 патерни AI-тексту

### Канцеляризми
1. Відіменні дієслова замість дієслів
2. Ланцюги родового відмінку
3. Уникання «це»/«є» (Copula Avoidance)
4. Пасивні конструкції замість активних
5. Іноземні слова там, де є українські

### Росіянізми та суржик
6. Типові росіянізми в AI-тексті українською
7. Кальки граматичних конструкцій

### AI-словник і роздування
8. Перевантажені слова (AI Vocabulary)
9. Роздування значущості
10. Поверхневий аналіз із дієприслівниками
11. Рекламна мова
12. Розмиті атрибуції (Weasel Words)
13. Шаблонні переходи
14. Формульні висновки про «виклики та перспективи»

### Структурні
15. Негативні паралелізми [ЖОРСТКА ЗАБОРОНА]
16. Правило трьох
17. Елегантна варіація (Synonym Cycling)
18. Хибні діапазони

### Тон і комунікація
19. Підлесливий тон (Sycophancy)
20. Артефакти чатбота
21. Хеджування (перестраховка)
22. Загальні позитивні висновки
23. Емодзі та надмірне форматування

## Жорсткі заборони

- «не просто X, а Y»
- «не лише X, але й Y»
- довге тире «—» (тільки коротке «-»)
- «являється» (росіянізм)
- риторичні запитання
